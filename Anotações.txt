Instalar dependencias do webpack
npm i --save-dev webpack@1.14.0 webpack-dev-server@1.16.2

Instalar dependencias do Babel

npm i --save-dev babel-core@6.22.1 babel-loader@6.2.10 babel-preset-es2015@6.22.0


Instalar plugin do babel spread para utilizar
npm i --save --dev babel-plugin-transform-object-rest-spread

Instalar React
npm i --save-dev react@15.4.2

Install preset react
npm i --save-dev babel-preset-react@6.22.0

Install para css
npm i --save-dev extract-text-webpack-plugin@1.0.1 style-loader@0.13.1 css-loader@0.26.1

Se executar o webpack veremos q na pasta public ele criou o app.css e o bundle.js
